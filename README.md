# 🍔 Food Delivery Time Prediction System

## Project Overview
This project predicts the estimated food delivery time using Machine Learning.

## Features
- Data Cleaning
- Exploratory Data Analysis (EDA)
- Linear Regression Model
- Delivery Time Prediction
- Streamlit Web Application

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn
- Streamlit

## Dataset
Food Delivery Dataset

## Model
- Linear Regression

## Run the Project

```bash
pip install -r requirements.txt
streamlit run app.py
```
